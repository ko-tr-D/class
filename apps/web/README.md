# Web

교사용 대시보드와 학생용 문제 풀이 화면을 둘 Next.js/React 앱 자리입니다.

브라우저에서 모든 핵심 업무가 가능하도록 설계합니다.

현재는 의존성 없이 바로 열어볼 수 있는 정적 프로토타입이 들어 있습니다.

## 현재 프로토타입

- `index.html`: 교사 이메일 로그인, 학생 접속 코드, 교사용 대시보드 첫 화면
- `styles.css`: 대시보드 UI 스타일
- `app.js`: 로그인 모드 전환과 대시보드 진입 흐름

## 주요 화면

- 교사용 대시보드
- 로그인
- 반/학생 관리
- PDF 자료함
- OCR 검수 화면
- 평가 기준 관리
- 학생별 기록 카드
- 생활기록부 문구 초안 검토
- 학생용 문제 풀이 화면
- 반별 성취도/풀이 패턴 분석 화면

## 화면 원칙

- 교사용 화면은 자료 정리와 반복 입력이 많으므로 조용하고 밀도 높은 대시보드로 설계합니다.
- 학생용 화면은 작은 기기에서도 풀 수 있게 단순하고 집중된 문제 풀이 화면으로 둡니다.
- 학생 개인정보, 평가 결과, 생활기록부 문구 화면은 권한 검사와 접근 로그를 전제로 합니다.
