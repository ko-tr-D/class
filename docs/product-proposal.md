# 프로그램 제안서

## 제품 이름 후보

- 국어수업 성장기록
- Class Learning Record
- 국어 성장 데이터북

## 제품 형태

설치형 프로그램이 아니라 웹브라우저에서 접속하는 웹앱입니다. 교사는 학교 PC, 개인 노트북, 태블릿에서 같은 자료를 열 수 있고, 학생은 개인 기기에서 접속 코드로 문제를 풉니다.

## 핵심 사용자

- 교사: 자료 업로드, 평가 기준 관리, 학생별 성취 확인, 생활기록부 문구 검토
- 학생: 개별 기기에서 문항 풀이, 자기 성취 확인
- 관리자/담임 확장 가능성: 학년도별 기록 조회, 반별 통계 확인

## 핵심 문제

중학교 국어 수업에서는 필기, 발표, 수행평가, 형성평가, 독서 활동, 글쓰기 결과물처럼 학생의 성장을 보여주는 자료가 많이 생깁니다. 하지만 이 자료들은 PDF, 종이, 엑셀, LMS 기록, 교사의 기억 속에 흩어지기 쉽습니다.

이 앱은 수업 자료를 데이터베이스화하고, 평가 기준에 맞춰 근거를 연결하며, 학생의 성장 특성이 드러나는 생활기록부 문구 초안을 교사가 더 쉽게 작성하도록 돕습니다.

## 주요 기능

### 1. 손글씨 필기 PDF 데이터베이스화

- PDF 업로드
- 페이지별 OCR 처리
- 문단/키워드/활동 단위 분리
- 교사 검수 화면
- 수업 날짜, 단원, 성취기준, 학생/반 연결

### 2. 평가 기준 기반 평가

- 평가 기준표 등록
- 기준별 수준, 근거, 피드백 문장 관리
- 학생별 활동 기록과 평가 기준 연결
- 반별/학생별 성취 흐름 확인

### 3. 생활기록부 문구 생성

- 학생별 관찰 기록, 평가 근거, 풀이 패턴, 성장 포인트 통합
- 과장 없이 교사 문체에 맞춘 문구 초안 생성
- 금지 표현, 민감 정보, 단정적 표현 필터링
- 교사 수정 이력 저장

### 4. 학생용 문제 풀이

- 교과서/평가 문항 업로드
- 문항 유형, 단원, 성취기준 태깅
- 학생별 접속 코드 또는 링크 제공
- 풀이 시간, 재시도, 오답 유형, 선택지 변경 기록 수집

### 5. 풀이 패턴 분석

- 학생별 약점 단원과 문항 유형 분석
- 반 전체 오답 집중 구간 확인
- 풀이 시간, 정답률, 선택지 변경, 주관식 표현 패턴 분석
- 교사와 함께 알고리즘 버전을 개선하고 기록

## MVP 제안

첫 버전은 `필기 PDF 업로드 -> OCR 결과 검수 -> 평가 기준 연결 -> 학생별 기록 카드 -> 생활기록부 문구 초안`까지 만드는 것이 좋습니다.

학생용 문제 풀이 기능은 데이터 구조를 처음부터 열어두되, 화면과 실시간 풀이 기능은 2단계에서 붙이는 편이 안정적입니다.

## 교사용 화면 초안

- 대시보드: 오늘 처리할 자료, 최근 평가, 학생별 주의/성장 신호
- 자료함: PDF, 교과서, 평가 문항 업로드 및 태깅
- 평가 기준: 단원별 성취기준과 평가 루브릭 관리
- 학생 기록: 학생별 활동, 평가, 문구 초안, 수정 이력
- 분석실: 반별 성취도, 풀이 패턴, 알고리즘 실험 기록

## 학생용 화면 초안

- 접속 코드 입력
- 배정된 평가/문항 목록
- 문항 풀이
- 제출 후 간단한 피드백

## 보안 방향

- 학생 기본정보와 학습/평가 기록을 분리합니다.
- 교사는 자신이 맡은 반과 학년도 자료만 볼 수 있습니다.
- 학생은 본인에게 배정된 평가 화면만 접근합니다.
- 생활기록부 문구와 관찰 기록은 민감 데이터로 분류합니다.
- AI에 보낼 데이터는 가능한 한 익명화하고, 최종 문구는 교사가 승인합니다.

## 중요 설계 원칙

- AI 결과는 최종 기록이 아니라 교사용 초안입니다.
- 생활기록부 문구는 항상 근거 자료와 연결되어야 합니다.
- 학생 개인정보와 평가 기록은 학년도, 반, 권한 단위로 분리합니다.
- 분석 알고리즘은 버전 관리하여 언제 어떤 기준으로 판단했는지 남깁니다.
